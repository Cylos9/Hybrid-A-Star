# Hybrid-A*
Motion Planner for a Car based of Kinematic & Reeds-Shepp Node expansions and a combination of Holonomic & Non-Holonomic Cost Function.

## Dependencies
1. python3
2. [CurvesGenerator](https://github.com/zhm-real/CurvesGenerator)
